## ARIAロールと属性

| コンポーネント | ロールと属性  | 使い方                                       |
| -------------- | ------------- | -------------------------------------------- |
| `Icon`         | `role="img"`  | 画像であることを示します。                   |
|                | `aria-hidden` | 要素をアクセシビリティツリーから除外します。 |
