:::note status=error
`Divider`は`v2.0`で削除される予定です。代わりに[Separator](/components/layouts/separator)を使用してください。
:::

## インポート

```ts
import { Divider } from "@yamada-ui/react"
```

## 使い方

```tsx
<Flex direction="column" gap="md">
  <For each={["solid", "dashed", "dotted"]}>
    {(variant, index) => <Divider key={index} variant={variant} />}
  </For>
</Flex>
```

### 垂直に区切り線を引く

```tsx
<Flex h="sm" gap="md">
  <For each={["solid", "dashed", "dotted"]}>
    {(variant, index) => (
      <Divider key={index} orientation="vertical" variant={variant} />
    )}
  </For>
</Flex>
```

:::note status=warning
区切り線を垂直にする場合は、親要素または`Divider`に高さを設定する必要があります。
:::
