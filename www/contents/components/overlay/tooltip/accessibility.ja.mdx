`Tooltip`は、アクセシビリティに関して[WAI-ARIA - Tooltip Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/tooltip/)に従います。

## キーボード操作

| キー     | 説明                                       | 状態 |
| -------- | ------------------------------------------ | ---- |
| `Escape` | 表示中のポップアップ要素を非表示にします。 | -    |

## ARIAロールと属性

| コンポーネント | ロールと属性     | 使い方                          |
| -------------- | ---------------- | ------------------------------- |
| `Tooltip`      | `role="tooltip"` | `Tooltip`であることを示します。 |
