`FormControl`を使用しない場合は、`Dropzone`に`aria-label`または`aria-labelledby`を設定します。

```tsx live=false
<Dropzone aria-label="Upload file">
  <Text>Drag file here or click to select file</Text>
</Dropzone>
```

```tsx live=false
<VStack gap="sm">
  <Text as="h3" id="label">
    Upload file
  </Text>

  <Dropzone aria-labelledby="label">
    <Text>Drag file here or click to select file</Text>
  </Dropzone>
</VStack>
```

## ARIAロールと属性

| コンポーネント          | ロールと属性          | 使い方                                                   |
| ----------------------- | --------------------- | -------------------------------------------------------- |
| `Dropzone`              | `role="presentation"` | プレゼンテーションであることを示します。                 |
| `input` <Tag>内部</Tag> | `type="hidden"`       | 要素をアクセシビリティツリーから除外します。             |
|                         | `aria-readonly`       | `isReadOnly`が設定されている場合は`"true"`を設定します。 |
|                         | `aria-disabled`       | `isDisabled`が設定されている場合は`"true"`を設定します。 |
|                         | `aria-invalid`        | `isInvalid`が設定されている場合は`"true"`を設定します。  |
|                         | `aria-required`       | `isRequired`が設定されている場合は`"true"`を設定します。 |
