`FileButton`にアイコンのみを使用する場合は、`FileButton`に`aria-label`を設定します。

```tsx live=false
<FileButton as={IconButton} icon={<PlusIcon />} aria-label="File upload" />
```

## ARIAロールと属性

| コンポーネント | ロールと属性    | 使い方                       |
| -------------- | --------------- | ---------------------------- |
| `FileButton`   | `role="button"` | ボタンであることを示します。 |
