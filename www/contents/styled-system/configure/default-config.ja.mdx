---
title: デフォルトのコンフィグ
description: Yamada UIのデフォルトのコンフィグ。
order: 1
---

デフォルトのコンフィグは、[カラーモード](/styled-system/color-mode)で使用される`initialColorMode`などが設定されています。

実際に設定されている値はこちらです。

```ts
const config: ThemeConfig = {
  initialColorMode: "light",
  disableTransitionOnChange: false,
  var: { prefix: "ui" },
  breakpoint: { direction: "down", identifier: "@media screen" },
}
```

| プロパティ                  |                        | 説明                                                                                                                                                                                                                                                           | デフォルト                                                                             |
| --------------------------- | ---------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------- |
| `initialThemeScheme`        |                        | [テーマを切り替える](/styled-system/theming/switch-themes)で使用されるテーマの初期値。                                                                                                                                                                         | `"base"`                                                                               |
| `initialColorMode`          |                        | [カラーモード](/styled-system/color-mode)の初期値。`light`, `dark`, `system`を受け入れます。                                                                                                                                                                   | `"light"`                                                                              |
| `disableTransitionOnChange` |                        | [カラーモード](/styled-system/color-mode)の切り替え時のトランジションなどによる不要な動きを避けるための設定。                                                                                                                                                  | `false`                                                                                |
| `locale`                    |                        | [Calendar](/components/data-display/calendar)や[FormatNumber](/components/other/format-number)で使用されるロケールの初期値。                                                                                                                                   | [navigator.language](https://developer.mozilla.org/ja/docs/Web/API/Navigator/language) |
| `breakpoint`                |                        | [レスポンシブスタイル](/styled-system/responsive-styles)で使用されるブレイクポイントの設定。                                                                                                                                                                   |                                                                                        |
|                             | `direction`            | [レスポンシブスタイル](/styled-system/responsive-styles)で使用されるブレイクポイントの方向を設定。`"up"`の場合は`@media(min-width)`を使ったモバイルファーストになり、`"down"`の場合は`@media(max-width)`を使ったPCファーストのレスポンシブスタイルになります。 | `"down"`                                                                               |
|                             | `base`                 | `direction`が`"down"`の場合のみ適用されるブレイクポイントの最大値。                                                                                                                                                                                            | `"9999px"`                                                                             |
|                             | `identifier`           | [レスポンシブスタイル](/styled-system/responsive-styles)で使用されるブレイクポイントのクエリを設定します。                                                                                                                                                     | `"@media screen" `                                                                     |
| `var`                       |                        | CSSの`カスタムプロパティ`に使用される設定。                                                                                                                                                                                                                    |                                                                                        |
|                             | `prefix`               | CSSの`カスタムプロパティ`に使用されるプレフィックスを設定します。                                                                                                                                                                                              | `"ui"`                                                                                 |
| `theme`                     |                        | [テーマのカスタマイズ](/styled-system/theming/customize-theme)で使用される設定。                                                                                                                                                                               |                                                                                        |
|                             | `responsive`           | テーマのトークンがレスポンシブオブジェクトに対応するかどうかを設定します。                                                                                                                                                                                     | `false`                                                                                |
| `loading`                   |                        | [ローディング](/styled-system/loading)で使用される設定。                                                                                                                                                                                                       |                                                                                        |
|                             | `screen`               | スクリーンローディングのオプションを設定します。                                                                                                                                                                                                               | `undefined`                                                                            |
|                             | `page`                 | ページローディングのオプションを設定します。                                                                                                                                                                                                                   | `undefined`                                                                            |
|                             | `background`           | バックグラウンドローディングのオプションを設定します。                                                                                                                                                                                                         | `undefined`                                                                            |
|                             | `custom`               | カスタムローディングのオプションを設定します。                                                                                                                                                                                                                 | `undefined`                                                                            |
| `alert`                     |                        | [Alert](/components/feedback/alert)で使用される設定。                                                                                                                                                                                                          |                                                                                        |
|                             | `statuses`             | [Alert](/components/feedback/alert)のステータスを設定します。                                                                                                                                                                                                  | `undefined`                                                                            |
| `status`                    |                        | [Status](/components/feedback/status)で使用される設定。                                                                                                                                                                                                        |                                                                                        |
|                             | `statuses`             | [Status](/components/feedback/status)のステータスを設定します。                                                                                                                                                                                                | `undefined`                                                                            |
| `notice`                    |                        | [Notice](/styled-system/notification)で使用される設定。                                                                                                                                                                                                        |                                                                                        |
|                             | `options`              | [Notice](/styled-system/notification)のオプションを設定します。                                                                                                                                                                                                | `undefined`                                                                            |
|                             | `variants`             | [Notice](/styled-system/notification)のバリアントを設定します。                                                                                                                                                                                                | `undefined`                                                                            |
|                             | `gap`                  | [Notice](/styled-system/notification)間のギャップを設定します。                                                                                                                                                                                                | `"md"`                                                                                 |
|                             | `appendToParentPortal` | ポータルが親ポータル内にある場合に、ポータルを親ポータルのノードに追加するかどうかを設定します。                                                                                                                                                               | `true`                                                                                 |
|                             | `containerRef`         | ポータルがアタッチされるコンポーネントの`ref`を設定します。                                                                                                                                                                                                    | `undefined`                                                                            |
|                             | `listProps`            | [Notice](/styled-system/notification)のリスト要素の`props`を設定します。                                                                                                                                                                                       | `undefined`                                                                            |
|                             | `itemProps`            | [Notice](/styled-system/notification)のアイテム要素の`props`を設定します。                                                                                                                                                                                     | `undefined`                                                                            |
| `snacks`                    |                        | [Snacks](/hooks/use-snacks)で使用される設定。                                                                                                                                                                                                                  |                                                                                        |
|                             | `options`              | [Snacks](/hooks/use-snacks)のオプションを設定します。                                                                                                                                                                                                          | `undefined`                                                                            |
|                             | `variants`             | [Snacks](/hooks/use-snacks)のバリアントを設定します。                                                                                                                                                                                                          | `undefined`                                                                            |
|                             | `gap`                  | [Snacks](/hooks/use-snacks)間のギャップを設定します。                                                                                                                                                                                                          | `"md"`                                                                                 |
|                             | `gutter`               | [Snacks](/hooks/use-snacks)の上下の間隔を設定します。                                                                                                                                                                                                          | `[0, 0]`                                                                               |
|                             | `negateMargin`         | `gutter`に負のマージンを適用するかどうかを設定します。                                                                                                                                                                                                         | `true`                                                                                 |
| `motion`                    |                        | [Motion](https://motion.dev/docs/react-motion-config)で使用される設定。                                                                                                                                                                                        |                                                                                        |
|                             | `config`               | [Motion](https://motion.dev/docs/react-motion-config)のオプションを設定します。                                                                                                                                                                                | `undefined`                                                                            |
