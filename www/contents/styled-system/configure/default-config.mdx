---
title: Default Config
description: Default config of Yamada UI.
order: 1
---

The default config is set with values such as `initialColorMode` used in [Color Mode](/styled-system/color-mode).

Here are the actual set values.

```ts
const config: ThemeConfig = {
  initialColorMode: "light",
  disableTransitionOnChange: false,
  var: { prefix: "ui" },
  breakpoint: { direction: "down", identifier: "@media screen" },
}
```

| Property                    |                        | Description                                                                                                                                                                                                                                                          | Default                                                                                |
| --------------------------- | ---------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------- |
| `initialThemeScheme`        |                        | Initial value of the theme used in [Switching Themes](/styled-system/theming/switch-themes).                                                                                                                                                                         | `"base"`                                                                               |
| `initialColorMode`          |                        | Initial value of the [Color Mode](/styled-system/color-mode). Accepts `light`, `dark`, `system`.                                                                                                                                                                     | `"light"`                                                                              |
| `disableTransitionOnChange` |                        | Setting to avoid unnecessary movements such as transitions during [Color Mode](/styled-system/color-mode) switching.                                                                                                                                                 | `false`                                                                                |
| `locale`                    |                        | Initial value of the locale used in [Calendar](/components/data-display/calendar) and [FormatNumber](/components/other/format-number).                                                                                                                               | [navigator.language](https://developer.mozilla.org/ja/docs/Web/API/Navigator/language) |
| `breakpoint`                |                        | Setting of breakpoints used in [Responsive Styles](/styled-system/responsive-styles).                                                                                                                                                                                |                                                                                        |
|                             | `direction`            | Set the direction of breakpoints used in [Responsive Styles](/styled-system/responsive-styles). In the case of `"up"`, it becomes mobile-first with `@media(min-width)`, and in the case of `"down"`, it becomes PC-first responsive style with `@media(max-width)`. | `"down"`                                                                               |
|                             | `base`                 | The base value for the `breakpoint` when `direction` is "down". This is treated as the largest `breakpoint`.                                                                                                                                                         | `"9999px"`                                                                             |
|                             | `identifier`           | Set the query of breakpoints used in [Responsive Styles](/styled-system/responsive-styles).                                                                                                                                                                          | `"@media screen" `                                                                     |
| `var`                       |                        | Setting used in CSS `custom properties`.                                                                                                                                                                                                                             |                                                                                        |
|                             | `prefix`               | Set the prefix used in CSS `custom properties`.                                                                                                                                                                                                                      | `"ui"`                                                                                 |
| `theme`                     |                        | Setting used in CSS [Customize Theme](/styled-system/theming/customize-theme).                                                                                                                                                                                       |                                                                                        |
|                             | `responsive`           | If `true`, theme tokens will accept responsive objects.                                                                                                                                                                                                              | `false`                                                                                |
| `loading`                   |                        | Setting used in [Loading](/styled-system/loading).                                                                                                                                                                                                                   |                                                                                        |
|                             | `screen`               | Set the options for screen loading.                                                                                                                                                                                                                                  | `undefined`                                                                            |
|                             | `page`                 | Set the options for page loading.                                                                                                                                                                                                                                    | `undefined`                                                                            |
|                             | `background`           | Set the options for background loading.                                                                                                                                                                                                                              | `undefined`                                                                            |
|                             | `custom`               | Set the options for custom loading.                                                                                                                                                                                                                                  | `undefined`                                                                            |
| `alert`                     |                        | Setting used in [Alert](/components/feedback/alert).                                                                                                                                                                                                                 |                                                                                        |
|                             | `statuses`             | Set the status of [Alert](/components/feedback/alert).                                                                                                                                                                                                               | `undefined`                                                                            |
| `status`                    |                        | Setting used in [Status](/components/feedback/status).                                                                                                                                                                                                               |                                                                                        |
|                             | `statuses`             | Set the status of [Status](/components/feedback/status).                                                                                                                                                                                                             | `undefined`                                                                            |
| `notice`                    |                        | Setting used in [Notice](/styled-system/notification).                                                                                                                                                                                                               |                                                                                        |
|                             | `options`              | Set the options for [Notice](/styled-system/notification).                                                                                                                                                                                                           | `undefined`                                                                            |
|                             | `variants`             | Set the variants for [Notice](/styled-system/notification).                                                                                                                                                                                                          | `undefined`                                                                            |
|                             | `gap`                  | Set the gap between [Notice](/styled-system/notification).                                                                                                                                                                                                           | `"md"`                                                                                 |
|                             | `appendToParentPortal` | Set whether to add the portal to the node of the parent portal when the portal is in the parent portal.                                                                                                                                                              | `true`                                                                                 |
|                             | `containerRef`         | Set the `ref` of the component where the portal is attached.                                                                                                                                                                                                         | `undefined`                                                                            |
|                             | `listProps`            | Props for [Notice](/styled-system/notification) list element.                                                                                                                                                                                                        | `undefined`                                                                            |
|                             | `itemProps`            | Props for [Notice](/styled-system/notification) item element.                                                                                                                                                                                                        | `undefined`                                                                            |
| `snacks`                    |                        | Setting used in [Snacks](/hooks/use-snacks).                                                                                                                                                                                                                         |                                                                                        |
|                             | `options`              | Set the options for [Snacks](/hooks/use-snacks).                                                                                                                                                                                                                     | `undefined`                                                                            |
|                             | `variants`             | Set the variants for [Snacks](/hooks/use-snacks).                                                                                                                                                                                                                    | `undefined`                                                                            |
|                             | `gap`                  | Set the gap between [Snacks](/hooks/use-snacks).                                                                                                                                                                                                                     | `"md"`                                                                                 |
|                             | `gutter`               | Set the vertical spacing of [Snacks](/hooks/use-snacks).                                                                                                                                                                                                             | `[0, 0]`                                                                               |
|                             | `negateMargin`         | Set whether to apply a negative margin to `gutter`.                                                                                                                                                                                                                  | `true`                                                                                 |
| `motion`                    |                        | Setting used in [Motion](https://motion.dev/docs/react-motion-config).                                                                                                                                                                                               |                                                                                        |
|                             | `config`               | Set the options for [Motion](https://motion.dev/docs/react-motion-config).                                                                                                                                                                                           | `undefined`                                                                            |
